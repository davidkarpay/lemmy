# Apollo Eagle Lunar Lander Simulator

This repository now holds the Lunar Module landing simulator project that began in the ChatGPT canvas.

Current state:
- A working canvas-based React prototype exists.
- The next intended patch line is v1.1, focused on authenticity repairs rather than scope expansion.
- The highest priorities are authentic procedure flow, realistic handling, and improved instrument fidelity.

Planned mission slice:
- Undocking from Columbia
- Station-keeping and inspection
- Compressed Descent Orbit Insertion
- Powered Descent Initiation
- P63 braking
- P64 approach
- P66 manual hover and touchdown

Modes:
- Arcade
- Realistic
- Authentic presentation
- Training presentation

This repository was initialized from the ChatGPT interface so development can continue outside canvas limits.
